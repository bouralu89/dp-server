dp-server
==========================

This is a node.js application build as a part of my diploma thesis.

## Requires

- NodeJS
- MongoDB

## Install it

	  $ git clone git://github.com/bouralu89/dp-server.git
	  $ cd dp-server
	  $ npm install
	  $ npm start 

**NOTE:** Do not forget to update your MongoDB URL in `config/config.js` before `npm start`.
